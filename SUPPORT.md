# Support

Support for this project is handled through the support portal:
- https://christruitt.com/support

If this project saves you time, you can support ongoing development:

- https://christruitt.com/tip-jar

Before submitting a request
- Confirm your Hubitat platform version.
- Confirm your installed app/driver version.
- Reproduce the issue once with debug logging enabled (if the project offers it), then capture logs.

Include this information
- Hubitat model and platform version
- App/driver version
- Steps to reproduce
- Expected behavior vs actual behavior
- Relevant logs (copy/paste)
- Screenshots of preference screens (if relevant)

Community support
If you are posting in the Hubitat community, do not share secrets (tokens, client secrets, API keys).
