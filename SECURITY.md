# Security Policy

## Supported versions
Security fixes are applied to the latest released version.

## Reporting a vulnerability
Report suspected security issues through the support portal:
- https://christruitt.com/support

Guidelines
- Do not open a public GitHub issue for security reports.
- Do not include secrets or personal data in your report.
- Include enough detail to reproduce (Hubitat platform version, app/driver version, steps, and a minimal log excerpt).
